## **Design & UI**

- **ThaiBev-inspired color scheme** with emerald green and gold accents
- **Professional dashboard layout** with statistics cards and tabbed navigation
- **Responsive design** that works on desktop and mobile devices
- **Intuitive file upload** with drag-and-drop functionality


## **Core Features**

### 1. **File Upload System**

- Support for CSV, XLS, and XLSX files (up to 5MB)
- Real-time validation of file format and required columns
- Progress tracking and error handling
- Batch processing with detailed feedback


### 2. **Delivery Dashboard**

- Real-time status tracking (Confirmed, Rescheduled, Pending, No Response)
- Advanced filtering and search capabilities
- Export functionality for reports
- Auto-refresh every 60 seconds


### 3. **Reports & Analytics**

- Performance summary with key metrics
- Daily breakdown reports
- Top reschedule reasons analysis
- CSV export for further analysis


## **Backend Integration**

### 4. **LINE Bot Integration**

- **Webhook handler** for processing customer responses
- **Flex Message templates** with Thai language support
- **Quick Reply buttons** for easy customer interaction
- **Signature verification** for security


### 5. **Scheduled Messaging**

- **Cron job endpoint** for 09:00 ICT message sending
- **Batch processing** (500 messages per batch to respect LINE limits)
- **Rate limiting** and error handling
- **Delivery confirmation tracking**


## **Technical Stack**

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** with custom ThaiBev theme
- **shadcn/ui** components
- **LINE Bot SDK** integration ready
- **Server Actions** for file processing
